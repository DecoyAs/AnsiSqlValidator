**Regular Expressions (RegEx)**



&nbsp;	Match SQL keywords and clauses

&nbsp;	Validate table and column names

&nbsp;	Check data types (INT, VARCHAR(n), etc.)

&nbsp;	Detect syntax errors like missing commas or parentheses

&nbsp;	Enables pattern-based validation using Python’s regex features.

&nbsp;	**RegEx** will be used to manipulate strings, since the sql queries will be in the form of strings.



**Only Python language features will be used, namely:**

&nbsp;	

&nbsp;	Regular Expressions (regex)

&nbsp;	String manipulation (upper(), split(), replace())

&nbsp;	Lists and dictionaries

&nbsp;	Functions and modular design

&nbsp;	Conditional logic and loops
